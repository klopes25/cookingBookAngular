# **Category**

*It is both a display and change component for the category property of a recipe*

## **inputs**
- **category: string** ("plat" by default)

*Displays the category value of the current recipe as an image and text*

- **edition: boolean** (false by default)

*If true, then arrows allowing to modify the value of the current category will appear*

----------
## **outputs**

- categoryUpdated: string

*This event is returned when the category value is changed. The new selected value is sent*

----------
## **Local variables**

*Nothing*
